# Stock Trading Dashboard

This is a modern stock watchlist dashboard built using HTML5, CSS, JavaScript, and Bootstrap. 
The application allows users to search stocks, add them to a watchlist, view stock charts, 
receive price alerts, and simulate live stock price updates.

## Features

- Stock search functionality
- Watchlist (favorite stocks)
- Highlight favorite stocks
- Sort stocks by price and change
- Simulated live stock price updates
- Stock price charts
- Price alert notifications
- Dark mode dashboard
- Responsive design using Bootstrap

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- Chart.js
